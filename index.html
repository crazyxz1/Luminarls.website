Comando: !luzes

Resgata as suas luzes.

Descrição: O comando !luzes Serve para Resgatar as luzes

Comando: !luz

Veja suas luzes ou outros usuários.

Descrição: O comando !luz permite verificar o saldo de luzes de outro usuário, apenas mencionando o nome do usuário após o comando.

Comando: !avatar

Mostra o avatar de um usuário.

Descrição: O comando !avatar exibe o avatar de qualquer usuário mencionado. É uma maneira fácil de visualizar a imagem do perfil de outros membros.

Comando: !banner

Mostra o banner de um usuário, se ele tiver um.

Descrição: O comando !banner exibe o banner de um usuário, caso ele tenha configurado um. Caso contrário, exibe uma mensagem informando que não há banner.

Comando: !pay

Envie luzes para outro usuário.

Descrição: O comando !pay permite que você envie uma quantidade de luzes para outro usuário. Basta mencionar o destinatário e a quantidade de luzes que deseja enviar.

Comando: /luzguia

Acessa o sistema de economia do bot.

Descrição: O comando /luzguia é uma funcionalidade do bot Luminarls que acessa o sistema de economia do bot. Ao utilizar esse comando, o usuário pode visualizar seu progresso no sistema de economia, acessar opções de interação com o bot e obter informações detalhadas sobre suas luzes, que são a moeda do bot.

Comando: /luminfo

Exibe informações sobre o bot.

Descrição: O comando /luminfo exibe informações gerais sobre o bot Luminarls, como a versão atual, uptime e outros dados importantes.
